Для сохранения изменений в общую базу нужен GitHub Personal Access Token с правом repo.
Токен хранится только в вашем браузере и не передаётся третьим лицам.

Получить токен: GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → New token → Repo: Contents (Read & Write)
